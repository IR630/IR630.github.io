# Портфолио Лабораторных Работ

Этот репозиторий содержит статический сайт-портфолио с отчетами по лабораторным работам.

## Технологии
*   **Генератор:** MkDocs
*   **Тема:** Material for MkDocs
*   **Хостинг:** GitHub Pages

## Обоснование выбора темы
Для сайта была выбрана тема **Material for MkDocs**, так как она:
1.  Обладает современным и "премиальным" дизайном из коробки.
2.  Поддерживает адаптивность (удобно смотреть с мобильных устройств).
3.  Имеет отличную поддержку темной темы и поиска.
4.  Широко используется в индустрии для технической документации.

## Структура проекта
*   `/src` — исходные файлы сайта (Markdown, конфигурация).
*   `/docs` — собранный статический сайт (публикуется на GitHub Pages).
