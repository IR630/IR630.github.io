# Лабораторная работа №6

> **Тема:** Предобработка данных. Исследовательский анализ датасета Titanic

## Цель работы
Освоить полный цикл предобработки табличных данных: первичный анализ, обработка пропусков, создание новых признаков (Feature Engineering), удаление выбросов и агрегация — на примере классического датасета о выживаемости пассажиров Титаника.

## Задание
1. Загрузить датасет и провести первичный анализ (типы признаков, распределения, статистики).
2. Обработать пропущенные значения в столбцах `Age`, `Embarked`, `Cabin`.
3. Выполнить Feature Engineering: создать новые признаки на основе имеющихся.
4. Обнаружить и устранить выбросы в числовых признаках.
5. Провести агрегацию данных и вычислить метрики качества очистки.

## Выполнение

### 1. Первичный анализ данных
Загружены обучающая и тестовая выборки. Выявлены признаки с пропусками:
- `Age` — пропуски заполнены **медианным значением** (28 лет).
- `Embarked` — пропуски заполнены **модой** (порт `S`).
- `Cabin` — признак полностью удален из-за высокой доли пропусков (>77%).

Построены гистограммы распределений для всех числовых признаков.

### 2. Feature Engineering (Создание признаков)
На основе исходных данных созданы три новых признака:

- **`Title`** — титул пассажира, извлеченный из поля `Name` (Mr, Mrs, Miss, Master и др.). Редкие титулы объединены в группу `Rare`, что позволяет отразить социальный статус.
- **`FamilySize`** — размер семьи: `SibSp + Parch + 1`.
- **`IsAlone`** — бинарный признак: 1, если пассажир путешествовал один, 0 — с семьей.

Также на основе возраста создан категориальный признак **`Age_group`** (Child / Teenager / Adult / Senior).

### 3. Удаление выбросов
- Для признака **`Fare`** применен **IQR-метод**: определены нижняя и верхняя границы, экстремальные значения заменены верхней границей (винзоризация).
- Для признака **`Age`** применено **обрезание по 95-му перцентилю**.

### 4. Агрегация и анализ
Рассчитана выживаемость в различных разрезах:
- По классу каюты (`Pclass`): пассажиры 1-го класса выживали значительно чаще.
- По полу и классу: женщины 1-го и 2-го класса имели наибольшую выживаемость.
- Построена сводная таблица выживаемости по `Title` и признаку `IsAlone`.

### 5. Корреляционный анализ
Для оценки связи между категориальными признаками применен **коэффициент V Крамера** (через `chi2_contingency` из `scipy.stats`). Тепловая карта корреляций подтвердила:
- **Пол (`Sex`) и Титул (`Title`)** — наиболее сильная связь с выживаемостью.
- **Класс каюты (`Pclass`)** — умеренная, но значимая связь.
