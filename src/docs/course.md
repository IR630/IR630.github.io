# Информация о курсе

## Программирование на Python (2 семестр)

Данный курс направлен на углубленное изучение языка Python и его применение в различных областях разработки и анализа данных.

### Основные темы курса:
1.  **Инструментарий разработчика:** Работа с Git, GitHub (SourceCraft), виртуальными окружениями и системами документации (MkDocs).
2.  **Научные вычисления:** Использование библиотеки NumPy для работы с массивами и матрицами.
3.  **Анализ данных:** Обработка табличных данных с помощью Pandas.
4.  **Визуализация:** Построение графиков и диаграмм с использованием Matplotlib и Seaborn.
5.  **Алгоритмы и структуры данных:** Реализация классических алгоритмов на Python.

### Цели обучения:
*   Освоение профессиональных инструментов разработки.
*   Получение навыков работы с большими объемами данных.
*   Автоматизация рутинных задач с помощью скриптов.
*   Подготовка качественной документации и отчетов.
